# meteostat.Granularity

Enumeration of data granularities available in Meteostat.

- **Type:** _Enumeration_
- **Source Code:** [GitHub](https://github.com/meteostat/meteostat/blob/main/meteostat/enumerations.py#L8)

The `Granularity` enumeration defines the different time intervals at which weather data can be retrieved from Meteostat. Each granularity represents a different temporal aggregation of meteorological data, from real-time observations to long-term climate averages. Different data sources and parameters may have varying availability across these granularities.

## Enumeration Members

- `HOURLY`
- `DAILY`
- `MONTHLY`
- `NORMALS`
